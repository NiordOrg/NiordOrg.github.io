Welcome to Niord - Nautical Information Directory

Editor and publication system for navigational warnings and notices to mariners (T&P).

Projects

  • niord: Source code for the base Niord web application
  • niord-dk: Customised Danish Niord overlay project.
  • niord-proxy: Simple website for displaying in-force NW & NM T&P from Niord.
  • niord-appsrv: Scripts for installing and setting up Wildfly and Keycloak for Niord.
  • niord-docker: Scripts for generating docker images for Niord.
  • docs.niord.org: Documentation for Niord. Published at docs.niord.org.
